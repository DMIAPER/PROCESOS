Página de Bienvenida

Esta aplicación web ha sido desarrollada como parte de la Tarea 5 del módulo de programación en Java.

El objetivo principal de esta tarea ha sido implementar un servidor HTTP básico utilizando Java puro, sin frameworks externos.

Se ha creado una estructura modular donde el servidor puede responder a diferentes rutas como /, /curiosidad y /contacto. Cada una de estas rutas muestra contenido HTML cargado desde archivos, de forma dinámica.

Además, se ha implementado un sistema sencillo para estructurar el contenido mediante plantillas HTML, con un encabezado, navegación, contenido principal y pie de página.

Este proyecto demuestra cómo se puede simular el comportamiento de un servidor web básico y servir contenido estático sin utilizar tecnologías avanzadas como servidores web profesionales o frameworks.

¡Gracias por visitar esta pequeña demostración!
